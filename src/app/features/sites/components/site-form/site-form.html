How to configure CSS selectors

CSS selectors identify elements on the page. Examples:

  • h1 — Selects all h1 elements
  • .price — Selects elements with the class "price"
  • #title — Selects the element with the id "title"
  • div.card a — Links inside divs with the class "card"
  • [data-price] — Elements with the attribute data-price

Tip: Use the browser's DevTools (F12) to inspect elements and copy selectors.
